Project Theseus

Created
2014
Length
5 Months
Type
Robotics Project
Role
Co-Creator

Description

The aim of this project was to tackle the various hardships associated with space exploration on a surface such as the Moon using virtual reality.

A rover was designed to face the harsh conditions of travelling across the Moon’s surface while being controlled through an interface designed to virtually allow you to drive from the perspective of the rover, effectively seeing what it sees.

A separate robot serves as a base-station to analyze samples of rocks collected by the robot based on their color and report findings back to the user, who controls these operations using an interface with gyroscopic sensors on two gloves and a special head-mounted screen to place user in the robot’s shoes.
