# Portfolio

Personal portfolio site showcasing projects across design, electronics, AI/ML, and full-stack development.

## Tech Stack

- [Next.js](https://nextjs.org/) (React, TypeScript)
- [React Three Fiber](https://docs.pmnd.rs/react-three-fiber) for 3D scenes
- [Tailwind CSS](https://tailwindcss.com/) for styling
- [Typed.js](https://mattboldt.com/demos/typed-js/) for typing animations

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
npm start
```
